### 📌 1. Bulk Assignment Update (Brightspace LMS)

Automates bulk updates of assignment (Dropbox) configurations in Brightspace LMS using API integration.

#### 🔧 Capabilities:
- Bulk update assignment properties
- Modify deadlines, instructions, and metadata
- Uses Brightspace LE APIs
- Reduces manual configuration effort

#### 📂 Input:
- Excel sheet with assignment IDs and update parameters

#### ⚠️ Notes:
- https://docs.valence.desire2learn.com/reference.html
- Requires API token configuration
- Designed for admin/instructor workflows


### 🎥 2. Bulk Recording Session Updater

#### 📂 Input:

#### ⚠️ Notes:
- https://docs.valence.desire2learn.com/reference.html
